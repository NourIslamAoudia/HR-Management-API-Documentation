Introduction

The HR Management API provides endpoints for managing employees, tasks, and human resources operations.

Authentication

POST /auth/login

Authenticate user and receive access token

Request Body: 
{
    "username": "string",
    "password": "string"
}
POST /add/admin

Create new admin account

Request Body: 
{
    "username": "string",
    "password": "string",
    "email": "string",
    "role": "string"
}
